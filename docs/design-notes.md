# Landing-page design notes

The physical reference is a shaded community gathering place at late afternoon: deep forest creates shelter, warm cream creates room to read, and terracotta/gold mark human action. The visual strategy is image-led and forest-drenched, not a generic charity card grid.

The hero is the signature motion moment. Programme stories reveal once as a connected sequence, while the sticky index makes the current theme legible. Pointer parallax is bounded to a decorative provisional motif. Touch devices receive no pointer effect and reduced-motion users receive a static experience.

Instrument Serif is restricted to “dignified life”; Inter carries every functional and institutional element. Programme media is thematic sample photography pending rights, consent and caption approval. It must never be described as IRI activity evidence.
