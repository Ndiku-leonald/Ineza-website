# Design system

## Direction

Calm editorial humanitarian design: forest green for trust, clay coral for warmth and action, ochre for hopeful highlights, and warm Ugandan earth neutrals. The provisional letterhead silhouette informed the human, grounded palette without being enlarged beyond its usable resolution.

## Tokens

- Primary `#173f35`; secondary `#286657`; accent `#d65c3f`; highlight `#e49a42`.
- Ink `#18231f`; muted `#5b6661`; cream `#f7f4ec`; warm `#ece4d4`; line `#d8d4c9`.
- Newsreader for display headings; Source Sans 3 for interface and body copy; system fallbacks included.
- Body 16–17 px; headings use a fluid 34–106 px scale. Comfortable line length is approximately 42–65 characters.
- Container: 76rem maximum; mobile gutter 1rem, tablet/desktop 2rem.
- Spacing follows a 4/8 px base; section spacing is fluid 64–120 px.
- Radius is deliberately restrained: 2 px controls and circular identity motifs only.

Buttons are solid and high-contrast; inline links remain underlined. Form fields use visible labels and explicit focus rings. Cards are reserved for comparable programme or governance content. Alerts use a coloured border, icon/text label and never colour alone. Images must have meaningful alternative text; decorative motifs use CSS or empty alt text. Layout is mobile-first with grid enhancement from 700 px and desktop navigation from 1050 px.

## Motion and photography

The homepage uses one orchestrated entrance, a lightly dimensional hero frame and pointer-responsive programme imagery. Movement is limited to transforms, opacity, clipping and bounded image scaling; touch devices do not run pointer tilt. All effects collapse to a static presentation under `prefers-reduced-motion`. Supplied photography is treated as thematic sample material only, never evidence of IRI delivery or participant identity.
