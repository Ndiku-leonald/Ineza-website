# Accessibility audit

Target: WCAG 2.2 AA.

Implemented: skip link; semantic landmarks; one primary page heading containing the organisation name; visible focus; keyboard-operable anchor navigation; Escape-to-close and focus restoration for mobile navigation; 44 px-class controls; reduced-motion handling; no audio; muted decorative video with poster fallback; no colour-only statuses; fluid text and layouts; meaningful photography alternatives; descriptive links.

Production browser checks cover navigation, menu behaviour, reduced motion and required widths. Remaining launch checks: test with NVDA/TalkBack, confirm final logo alternative text, review any future forms/payment widgets, and audit final image licences, consent and captions.
