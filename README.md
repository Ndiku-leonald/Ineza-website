# Ineza Refugees Initiative website

Premium single-page stakeholder-review website for IRI, built with Vite, React, strict TypeScript, Tailwind CSS and Lucide icons.

## Development

Use Node 20+ and npm. Run `npm install`, then `npm run dev`. Production validation: `npm run format`, `npm run lint`, `npm run typecheck`, `npm test`, `npm run build`, and `npm run test:e2e`.

Structured public content is in `src/data`. Page sections are in `src/components`; motion/accessibility utilities are in `src/hooks`. Source Word documents remain unchanged under `docs/source`.

Local photographs under `public/images` are thematic design samples. Replace them only with approved, licensed media and retain accurate alternative text. Optional muted programme videos can be added under `public/media` using the filenames documented in the brief; posters remain the graceful fallback.

No payment system or unverified achievement is implemented. The organisation has confirmed its domain and email; safeguarding, privacy, donation and image permissions still require approval before deployment. See `docs/content-review.md` and `docs/deployment.md`.
